Physics Based Resistor - Current Based

 

Block Description

This block is a current based resistor, adjusting resistance to achieve a desired voltage drop at a given current.

Mask Parameters

N/A

T_in Ambient Temperature
Resistance "Resistance Independent variable"

Run Script Inputs: Degradation Input ( > 0.0 )

Run Script Inputs: Degradation-Driving Parameter Adjustment ( < 1.0 )

resI_resistance_adjustment Parameter Adjustment for Internal Resistance
